# BoboSokoban
波波推箱子（java窗口体程序）Eclipse项目

经典小游戏《推箱子》


<h5>游戏规则</h5>
<h5>1.上下左右控制灰太狼移动，所有喜洋洋都关进笼子胜利。</h5>
<h5>Tips:当你不小心走错路按Enter可以撤回一次。</h5>
<h5>Tips:按Esc键可以在你所在的关卡重新开始。</h5>
